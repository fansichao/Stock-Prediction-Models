# 股票模型预测

tags: `202108` `机器学习` `股票预测`

## 环境说明

- Python 3.6
- 安装说明 `pip install -r requirements.txt`
- 运行说明 `python stock_prediction.py`

## 代码结构说明

```bash

- data                    数据文件
- run_res                 运行结果
- models.py               相关模型
- stock_prediction.py     股票预测代码
- tools.py                通用工具
```
